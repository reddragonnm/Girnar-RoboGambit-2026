## Game.py

- This file contains the starter code of Game Engine logic that would play chess.
- Use Models folder to maintain repository structure in case using any models.
- game.py contains default parameters and variables, helper function and main logic funtion.
- DO NOT CHANGE values of default parameters.
- get_best_move serves as the primary function and I/O format of the same must NOT be altered.
- Rest functions act as helper functions and user is independent to alter them as per their use.
